JSON 포맷터란?

JSON(JavaScript Object Notation)은 데이터를 주고받을 때 가장 많이 사용되는 형식입니다. 하지만 서버 로그나 API 응답값은 보통 공백 없이 뭉쳐져 있어(Minified) 사람이 읽기 어렵습니다.

이 도구를 사용하면 뭉쳐진 JSON 코드를 들여쓰기가 적용된 예쁜 형태(Prettified)로 변환하여 쉽게 분석할 수 있습니다. 또한, 문법 오류가 있는 경우 어디가 틀렸는지 알려주는 검증(Validation) 기능도 포함되어 있습니다.

사용 방법

  • 정렬 (Beautify): 왼쪽 입력창에 JSON을 넣고 버튼을 누르면 읽기 좋게 변환됩니다.
  • 압축 (Minify): 불필요한 공백을 모두 제거하여 용량을 줄입니다.
